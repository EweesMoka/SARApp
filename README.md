## SARApp功能
我在Qt学习中做的小东西，主要功能如下：
1. 包括接收端和发送端两个界面，采用udp组播实现定时收发数据；
2. 在接收端以表格的形式展现收到的数据，采用model/view，并且按照收到的时间作升序降序；
3. 10s内若没有新数据进来，清空所有数据；
4. 做一个自定义滑动开关，控制是否接收数据；
5. 做白天黑夜两种皮肤，实现换肤功能；
